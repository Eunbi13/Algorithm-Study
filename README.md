### 규칙
> 1. 클래스 파일만 올린다.
> 2. 문제당 1시간을 넘지 않도록 하며 안풀리는 경우 다음 문제로 일단 넘어가거나 답안을 참고한다.
> 3. 패키지명은 ps{문제번호}으로 작성한다. ex) ps2557
> 4. 클래스명은 이름(닉넴)으로 작성한다. ex) Eunbi.java
